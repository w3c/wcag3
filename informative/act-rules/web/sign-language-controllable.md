The rule checks that sign language can be turned on and off. 

## Applicability

This rule applies to any synchronized media target that includes visible sign language interpretation within the page/view.

## Expectation

The test target has a control for the user to turn the sign language on or off. Except when the sign language is embedded or a part of the visible video itself.

## Examples

### Passed example 1

A video that contains sign language is able to be turned on or off.

### Failed example 1

A video that contains sign language is not able to be turned on or off.
