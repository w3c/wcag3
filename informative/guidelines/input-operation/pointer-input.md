## User needs

* As a person with a motor disability, I need to be able to use all functionality without requiring fine pointer control.
* As a person with a motor disability, I need controls that do not require precise pressure to operate.
* As a person with a motor disability, I need to be able to use all functionality without having to tilt a pointing device in a specific way.
* As a person with a motor disability, I need to be able to use all functionality without having to move a pointer at a particular speed.
* As a person with a motor disability, I need to be able to use all functionality with a single pointer rather than requiring multi-pointer interactions.
* As a person with a motor disability that includes limited kinaesthetic perception, I need to be able to control the view without having to accurately position or orient a stylus.
