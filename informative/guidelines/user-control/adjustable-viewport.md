## User needs

* As a person with a limited field of view, I need to be able to reduce the width of my reading area so that I can see and track text more easily.
* As a person with low vision or a vestibular disorder, I need to be able to read content without moving my head.
* As a person with low vision, I need to be able to resize the entire presentation of content, including text, graphics, and video.
* As a person with low vision, I need to minimize scrolling in the direction of the rendered text.
* As a person with central vision loss, I need dynamic horizontal scrolling.
* As a person with low vision, I need to be able to reformat multi-column text into a single continuous reading flow.
* As a person with low vision, I need navigation features to remain available and easy to find when I zoom the interface.
*  As a person with a limited field of view, I need to be able to adjust the width and height of my reading area so that I can follow text from one line to the next.
* As a person with low vision, I need interface elements not to obscure the content I am viewing so that I can see, understand, and interact with it.
* As a person with low vision, I need to be able to hide or minimize toolbars, sidebars, and other non-essential interface elements so that I can focus on the content.
* As a person who uses a device mounted in a fixed orientation, I need digital content to work without requiring me to rotate my device.
* As a person with a cognitive or learning disability, I need to be able to adjust my view to minimize distractions and focus on the content that is relevant to my current task.
* As a person with a motor disability, I need to be able to complete critical tasks without having to move my head because the content spans a large visual field or uses mixed orientations.
