## User needs

* As a person with low vision, I need to be able to customize the text and background colors across the full range of colors supported by my technology.
* As a person with low vision, I need to be able to adjust text spacing between letters, words, lines and paragraphs.
* As a person with low vision, I need to be able to change the font family.
* As a person with low vision, I need content to flow in a way that supports continuous reading without unnecessary scrolling between columns.
* As a person with low vision, I need to be able to turn hyphenation on or off.
* As a person with low vision, I need to be able to reformat multi-column text into a continuous reading flow across the full page width.
* As a person with a cognitive or learning disability, I need semantic text indicators to reflect their actual purpose rather than being used only for visual effect.
* As a person with low vision, I need to be able to change the capitalization of text, including text originally presented in all capital letters.
* As a person with low vision, I need to be able to change the justification of blocks of text.
* As a person with low vision, I need to be able to customize the presentation of text, including justification, line spacing, and paragraph margins.
* As a person with low vision, I need to be able to customize the text and spacing of controls.
* As a person with low vision, I need to maximize the amount of magnified text that’s visible at one time so that I can minimize scrolling.
* As a person with dyslexia, I need to be able to customize the appearance of text at the line, sentence, or paragraph level.
