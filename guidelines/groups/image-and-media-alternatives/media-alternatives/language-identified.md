title: Speaker language identified
---

When more than one language is spoken in :term[audio] :term[content], the language spoken by each speaker is programmatically determined in all :term[media alternatives].

:::ednote
Is it better to say "When more than one language is spoken in :term[audio] :term[content], Portion(s) using a language different from the original language is programmatically determined in all :term[media alternatives]".
:::

:::except-when
Incidental word usage.
:::

:::tests
<i>Language identified in transcripts</i>

<b>Procedure</b>
1. Identify all audio or video transcripts within the unit of conformance that include multiple languages.
2. Portion(s) using a language different from the original language is programmatically determined in the media alternatives.

<b>Expected results</b>
* #2 is true.
:::
