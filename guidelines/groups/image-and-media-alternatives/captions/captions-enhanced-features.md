Enhanced features that allow users to interact with :term[captions] are available.

:::ednote
* Needs additional research.
* Need to define "enhanced".
* User needs:
  * As a person with a cognitive disability who is deaf or hard of hearing, I need the ability to use enhanced captions to interact with the captions, such as looking up key terms or reviewing additional visual information, such as  a labeled diagram.
* Method: Provide enhanced features that allow users to interact with captions 
:::

:::example
A link to review a glossary term or a labeled diagram.
:::

:::tests
<b>Procedure</b>
For each media asset with audio content that provides enhanced features to allow users to interact with captions:
1. Play the media with captions on.
2. Check that a mechanism is available so that users can interact with the captions using all relative input methods.

<b>Expected Results</b>
* #2 is true
:::
