<div class="issue" data-number="472"></div>

Speakers are identified understandably within all :term[media alternatives].

:::note
* Full name then use part of name
* Hidden identity as part of narrative structure
* Recognizable position within the context of use
:::

:::applies-when
There are multiple speakers in the video.
:::

:::example
Initially using "Maya Angelou" within the context of a story regarding poetry and then using "Angelou".
:::

:::tests
<i>Consistent speaker name in transcript</i>

<b>Procedure</b>
1. Check for any speakers in the audio or video content.
2. Check that each speaker in the audio or video is consistently identified in the transcript.

<b>Expected results</b>
* #2 is true.
:::
