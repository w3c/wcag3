:term[Content] does not include non-essential :term[flashing].

:::ednote
Method(s)
* Consider if flashing is essential and, if it is not, refrain from including it.
:::

:::tests
<b>Procedure</b>
1. Identify if content contains flashing.
2. For each instance of flashing, identify if the flashing is essential.

<b>Expected Results</b>
* #1 is false, or
* #2 is true
:::
