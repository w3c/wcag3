title: Path-based gesture not relied on
---

:term[Path-based gestures] are not the only way of achieving any functionality.

:::except-when
- A path-based gesture is :term[essential] to the functionality.
:::

:::tests
**Procedure**

For each path-based gesture:
1. Check that the functionality is available without a path-based gesture.

**Expected results**
- #1 is true
:::
