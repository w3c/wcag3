We assert that :term[content author(s)] follow user interface design principles that include minimizing the difference between the number of input commands required when using the :term[keyboard interface] only and the number of commands when using other input modalities.

:::note
Other input modalities include :term[pointing devices], voice and speech recognition, :term[gesture], :term[camera input], and any other means of input or control.
:::

:::
