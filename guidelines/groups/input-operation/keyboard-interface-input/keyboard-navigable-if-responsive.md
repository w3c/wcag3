If the :term[page] / :term[view] uses responsive design, the page / view remains fully keyboard navigable.

:::tests
- Carry out all keyboard input tests with the responsive content at all breakpoints defined by the author and at minimum width.
- PASS: all content at each user-defined breakpoint is fully keyboard navigable
- FAIL: Any content at any user-defined breakpoint isn't fully keyboard navigable
:::
