All :term[components] on the :term[page]/:term[view] that can be operated by :term[pointer], :term[audio] (voice or other), :term[gesture], camera, or other means can be operated using :term[keyboard interface] only.

:::tests

**Procedure**
For each interactive element:
1. Check that it is an be operated using only the keyboard and keyboard actions in the [Standard Keyboard Navigation &amp; Operation Keys and Techniques](https://github.com/w3c/wcag3/wiki/Standard-Keyboard-Navigation-&-Operation-Keys-and-Techniques) or described on the page where it is required or on a page earlier in the process where it is required.

**Expected results**
- #1 is true
:::
