The :term[keyboard focus] moves sequentially through :term[content] in an order and way that preserves meaning and operability.

:::tests
**Procedure**

1. Determine the logical order of the interactive elements.
2. Starting with the first focusable elemnent in the page/view, tab through the interactive elements in the content.
3. Check that the focus order of the interactive elements in the content follows is the same as the logical order in #1.

**Expected results**
- #3 is true
:::
